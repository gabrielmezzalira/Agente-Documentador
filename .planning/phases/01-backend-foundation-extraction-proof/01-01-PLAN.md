| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-01 | Tampering | POST /projects body | mitigate | Pydantic v2 ProjectCreate validates types; missing name/client → 422 (ASVS V5) |
| T-01-02 | Information Disclosure | SUPABASE_SERVICE_KEY in repo | mitigate | .env in .gitignore; only .env.example committed (no real values) |

| T-01-05 | Information Disclosure | response.data not checked → silent failure | mitigate | Every .execute() guarded with `if not response.data` (Pitfall 5) — surfaces FK/RLS errors as HTTP errors not false success |
| T-01-SC | Tampering | pip installs (14 packages) | mitigate | All packages [ASSUMED] in RESEARCH.md Package Legitimacy Audit; all are major multi-year ecosystem packages verified present on PyPI. See note below. |
